GWT 揭秘
-------------
	第六章, GWT控件详解

技术点
-------------
	1.界面对象(com.google.gwt.user.client.ui.UIObject)
	2.控件(com.google.gwt.user.client.ui.Widget) = 界面对象 + 生命周期管理 + 事件管理
	3.面板(com.google.gwt.user.client.ui.Panel) = 控件 + 布局管理
	4.控件--生命周期管理 (Detach, Attach, 状态切换流程)
	5.控件--事件管理 (事件类层次, 事件处理器, 事件分发流程)
	6.GWT全局类(GWT, DOM, Window)
